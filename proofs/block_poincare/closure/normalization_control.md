# Normalization Control Under Scaling

## Sequence
\[
\|f_L\| = 1.
\]

## Block rescaling
Let \(g_L\) be the rescaled functions on unit blocks.

## Measure normalization
\[
\|g_L\|^2 = \sum_B \int_B |g_L|^2.
\]

## Preservation
\[
\|g_L\| = 1.
\]

## Strong limit
\[
g_L \to g \quad \text{in } L^2.
\]

## Limit normalization
\[
\|g\| = \lim_{L\to\infty} \|g_L\| = 1.
\]

## Status
Normalization preserved in scaling limit.
